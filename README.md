# Days Reminder - 倒数日历

A simple, clean day counter app built with Flutter + Riverpod + Hive. Track important events and count down to special dates. Follow the structure in `Days_Reminder_Cursor_Prompt.txt`.

## Getting Started

Run:

```
flutter pub get
flutter run
```
